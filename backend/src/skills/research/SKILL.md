# Research

Multiple research capabilities:
- **Anaxa** — Ask the web about highlighted text (Gemini + Google Search)
- **Cerydra** — Document-grounded Q&A over client files (Gemini)
- **Hysilens** — Identify the source document for a snippet (Gemini)
- **Mydei** — ABM target research with ICP filtering (Gemini + Google Search)
