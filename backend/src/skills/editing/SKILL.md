# Editing

Rewrite LinkedIn post drafts while preserving 100% of the factual payload. Supports:
- Single post rewrite with style instructions
- Batch rewrite (up to 12 posts) with cross-post coordination
- Surgical edits (apply specific suggestions without full rewrite)
- Iterative rewriting for large draft sets
